# Instalação

1. Instale o [pip]([https://pip.pypa.io/](https://pip.pypa.io/en/stable/getting-started/));
2. Crie um ambiente virtual [venv](https://docs.python.org/3/tutorial/venv.html). No IntelliJ têm plugins que fazem isso mais automaticamente na criação do projeto, no VSCode o processo tende a ser mais manual;
3. Depois de criar e ativar o ambiente virtual _venv_, usar o comando:
	``` bash
	python -m pip install Django ## instala a versão mais recente do Django.
	
	python -m pip install Django==5.2 # instala a versão 5.2 do Django.
	```
